class pip_services_runtime.commands.CommandSet.CommandSet¶

Bases: object

Handles command registration and execution. Enables intercepters to control or modify command behavior

add_command(command)¶

Adds a command to the command set.

Args:
command: a command instance to be added

Returns: None

add_command_set(commands)¶

Adds commands from another command set to this one

Args:
commands: a commands set to add commands from

Returns: None

add_commands(commands)¶

Adds a list of commands to the command set

Args:
commands: a list of commands to be added

Returns: None

add_intercepter(intercepter)¶

Adds intercepter to the command set.

Args:
intercepter: an intercepter instance to be added.

Returns: None

execute(command, correlation_id, args)¶

Execute command by its name with specified arguments.

Args:
command: the command name. correlation_id: a unique correlation/transaction id args: a list of command arguments.

Returns: the execution result.

Raises:
MicroserviceError: when execution fails for any reason.
find_command(command)¶

Find a specific command by its name.

Args:
command: the command name.

Returns: found ICommand or None

get_commands()¶

Get all supported commands Returns: ICommand list with all commands supported by component.

validate(command, args)¶

Validates command arguments.

Args:
command: the command name. args: a list of command arguments.

Returns: MicroserviceError list of validation errors or empty list when arguments are valid.
