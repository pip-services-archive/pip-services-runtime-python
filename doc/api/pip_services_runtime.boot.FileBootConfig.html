class pip_services_runtime.boot.FileBootConfig.FileBootConfig¶

Bases: pip_services_runtime.boot.AbstractBootConfig.AbstractBootConfig

Boot configuration reader that gets microservice configuration from JSON file on local disk.

This is the simplest possible configuration. However, in large scale deployments it may be unpractical. The distrubuting configurations from a centralized configuration repository may be a better option. Check other types of readers to support those scenarios.
